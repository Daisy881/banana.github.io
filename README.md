# banana.github.io
一、简介
1.这是一个原生的微信小程序，所有页面均已开发完毕，但还没有添加后台以及数据库相关的东西，所以还只能算是一个不完成的半成品。
2.页面主要包括首页、广场、我的三个tabBar，其中包括搜索、详情以及广场上的游记发布和收藏等功能。
